이 프로그램은?
보배드림 사이트의 차단기능의 인원수 제한으로 인해 넘쳐나는 사용자를 관리할수 없기에..이기능을 구현함

#설치방법
1. 다운받아 압축파일을 임의폴더에 압축을 풀어서 크롬의 확장설정에 들어가 개발자 모드를 선택하시고 "압축해재된 확장 프로그램을 로드합니다." 를 이용해 압축을 푼 폴더를 선택하시면 활성화됩니다.
